# Real Sample Photos

> 這個資料夾放使用者實際手機拍的照片，作為去背升級的測試樣本。

## 命名規範

```
{difficulty}_{type}_{nn}.{ext}
```

範例：
- `easy_clothes_01.jpg` — 簡單衣服（單色背景、平放）
- `medium_clothes_01.jpg` — 中等難度衣服（衣架 + 家具背景）
- `hard_clothes_01.jpg` — 困難衣服（複雜背景）
- `dark_clothes_01.heic` — 暗光環境
- `pattern_clothes_01.jpg` — 有花紋 / Logo / 拉鏈

## 重要

- 請用 **iPhone 原檔**（USB / AirDrop / Google Drive 原檔下載 / Telegram 文件）
- **不要** Line / WeChat / WhatsApp（會壓縮）
- 不要截圖（會去除 EXIF）

## 排除 git

這個資料夾的 `*.jpg` `*.heic` `*.png` 已加入 `.gitignore`，照片不會上 git。

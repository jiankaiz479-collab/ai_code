⚠️ 以下是骨架草稿，等更接近推甄時補完。

## 自傳骨架

### 第 1 段：我是誰，為什麼想念這個所
- 學校 / 系 / 主修方向
- 為什麼選資工/AI 所（**具體一點**，不要寫「對 AI 有興趣」）
- 一個觸發點故事（例如：什麼時候開始想做 AI 應用）

### 第 2 段：學習軌跡
- 修過的相關課程（列 2~3 個關鍵）
- 自學的部分（線上課程、論文閱讀）
- 第一個真正動手做的 AI 專案 → **連到 Virtual Try-On 專題**

### 第 3 段：Virtual Try-On 專題（**主菜**）
> **這段直接從 portfolio 其他檔案抓素材整合**

關鍵要講的點：
- **問題定義**：線上服飾退貨率、為什麼 AI 是解法 → 引用 [01_problem.md](01_problem.md)
- **技術整合**：三個異質 AI（rembg + Gemini + Tripo） → 引用 [03_ai_choices.md](03_ai_choices.md)
- **工程細節**：6 段錯誤碼設計、Mock 模式 → 引用 [05_error_design.md](05_error_design.md)
- **成果**：3D 重建 35s、Mock 模式提速 70 倍 → 引用 [07_metrics.md](07_metrics.md)
- **遇到的困難**：紅衣變橘 / 積分燒完 → 引用 [06_bugs_and_fixes.md](06_bugs_and_fixes.md)

### 第 4 段：能力證明
- 我會什麼（具體技術，不要寫「精通」）
- 我的工作方式（會踩坑、會記錄、會 prompt engineering、會做成本控制）
- **我不會的我會誠實寫**（例如：理論深度有待加強、想補強深度學習數學）

### 第 5 段：為什麼是這間學校 / 這位老師
- 老師的研究方向 → **跟我做過的事的連結**
- 為什麼他能教我我目前還不會的東西
- 我希望貢獻什麼

## 讀書計畫骨架

### 短期（碩一）
- **補強理論基礎**：深度學習數學（線代 / 機率 / 最佳化）、生成模型理論（Diffusion / Score-based / VAE）
- **修課**：機器學習、深度學習、計算機視覺、生成模型專題（依老師領域微調）
- **延續專題**：把 Virtual Try-On 從工程 demo 升級成「研究等級」——加入評估指標、消融實驗

### 中期（碩二上）
- **研究方向**：在以下三個方向選一個深入
  1. **物理引導的神經網路布料模擬** — 把 Tripo 出的 3D 模型送進可學習物理模擬，學「真實布料垂墜」
  2. **多模態 Prompt 自動優化** — 從失敗 case 自動修 prompt（DSPy / RLAIF 方向）
  3. **單圖 3D 重建的紋理對齊** — 解決「紅衣變橘」這類紋理偏移問題的學術版本
- **目標**：寫出第一篇 workshop paper 或會議短文

### 長期（碩二下 / 畢業）
- 完整論文（如目標方向有產出）
- 工程能力 + 學術能力雙軌畢業

## 寫自傳的 3 條鐵則

1. **不要堆形容詞**：「強烈興趣」「無比熱忱」全部刪掉。用**事實**讓教授自己得出結論
2. **故事 > 條列**：「我做了 A、B、C」不如「為了解決 X，我做了 Y，過程中發現 Z」
3. **誠實寫弱點**：寫一個你不會的東西 + 你打算怎麼補 → 教授會更相信你的「會」
